简化请求运行时权限流程，AndPermission支持在Android 5.0（Api级别21）及更高的系统中请求权限，在比Android 5.0低的系统中它会直接返回成功。

* 在哪里使用
  * Activity
  * Fragment
  * SupportFragment(在support库中的)
  * Context

* 支持多重方式
  * 单个权限
  * 单个权限组
  * 多个权限
  * 多个权限组
